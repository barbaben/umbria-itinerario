# Itinerario Umbria di Betto e Sandra - GitHub Pages

Questo pacchetto è pronto per essere pubblicato su **GitHub Pages**.

## File inclusi
- `index.html` → pagina principale con itinerario dettagliato e pulsante verso la mappa
- `mappa.html` → pagina separata con la mappa interattiva

## Pubblicazione
1. Crea un repository GitHub, ad esempio `itinerario-umbria-betto-sandra`.
2. Carica nella root del repository i file `index.html` e `mappa.html`.
3. Vai su **Settings → Pages**.
4. In **Build and deployment** imposta:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Salva e attendi la pubblicazione.

## URL finale
Il sito sarà disponibile a un indirizzo simile a:
`https://TUO-USERNAME.github.io/itinerario-umbria-betto-sandra/`

## Uso in Notion
Una volta pubblicato il sito, puoi:
- usare il link principale `index.html` come pagina itinerario;
- incorporarlo in Notion tramite blocco `/embed`.

Se GitHub Pages pubblica dalla root, l'URL base aprirà direttamente `index.html`.
